Litigant Portal

Component Library

Living documentation for Cotton components with Tailwind CSS. These components are available to all templates via <c-componentname> syntax.

We use Django Cotton for components, Alpine.js for interactivity, and Tailwind CSS for styling.

{# ========== BUTTONS ========== #}

Buttons

Accessible button component with multiple variants. WCAG 2.1 AA compliant.

{# Demo #}

Demo

Primary Outline Dark Ghost Danger
Small Default Extra Large
Disabled
{# Props #}

Props

variant "primary" | "outline" | "dark" | "ghost" | "danger". Default: "primary"
size "sm" | "" | "xl". Default: "" (normal size)
type "button" | "submit" | "reset". Default: "button"
disabled
(optional) Boolean. Disables the button.
full_width
(optional) Boolean. Makes button full width.
{# Code #}
Show code 
<c-button variant="primary">Primary</c-button>
<c-button variant="outline">Outline</c-button>
<c-button variant="dark">Dark</c-button>
<c-button variant="ghost">Ghost</c-button>
<c-button variant="danger">Danger</c-button>

<c-button variant="primary" size="sm">Small</c-button>
<c-button variant="primary" size="xl">Extra Large</c-button>

<c-button variant="primary" disabled>Disabled</c-button>
